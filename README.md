# Expense Tracker

## Description
Expense Tracker is a web application designed to help users track their expenses, income, and savings. It provides user account functionality for personalized tracking and generates statistics to visualize financial data. The backend is powered by Supabase for secure data storage and management.

## Table of Contents
- [Features](#features)
- [Screenshots](#screenshots)
- [Technologies Used](#technologies-used)

## Features
- Track expenses, income, and savings.
- User authentication and account management.
- View detailed statistics and charts using Recharts.
- CSS styling with Tailwind CSS.
- Efficient data fetching and caching with React Query.
- Navigation between pages handled by React Router.
